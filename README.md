# CraftCMS 2.6 Boilerplate

A boilerplate setup for CraftCMS 2.6, including the CMS, ContentBuilder matrix fieldtype, TWIG layout templates, and CSS framework.


---

Contributing
======

Please read the [contribution guidelines] before submitting a pull request.

In particular: <strong>If you have commit access, please don't merge changes without
waiting a week for everybody to leave feedback.</strong>
